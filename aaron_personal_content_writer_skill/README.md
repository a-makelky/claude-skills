# Aaron Personal Content Writer Claude Skill

> Last updated: 2026.3.9

This package is optimized for Claude's "Upload Skill" feature.

## How to use

1.  **Keep this zip file as is** (or use the `.claude.skill.zip` extension).
2.  In Claude.ai, click on **Upload Skill**.
3.  Drag and drop this zip file.
4.  Claude will automatically parse the `SKILL.md` file for instructions and use `style_guide.md` for context.

### Manual Setup (Alternative)
If you prefer manual setup:
- Use `SKILL.md` as your **Custom Instructions**.
- Upload `style_guide.md` to your **Project Knowledge**.
